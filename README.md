# Fake News Detection (Colab-ready)

This project detects fake news using **Logistic Regression (TF-IDF)** and **LSTM (Keras)**.

## Features
- Text cleaning and preprocessing
- TF-IDF + Logistic Regression model
- Bidirectional LSTM neural network
- Ensemble prediction (average of models)
- Models saved automatically in `models_output/`
